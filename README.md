# 🎮 Play Fetch
This is a game made with Unity where you send dogs to catch balls falling from the sky. The challenge of this game is to time your dogs well to perfectly catch the balls. Additionally, the balls have different speed values depending on their color.

---

## 🛠️ Technical details
* **Spawn systems** -> Handles the random spawning patterns for the falling balls and instantiates a dog whenever the player presses SPACE.
* **Catch system** -> Registers successful object collection when a dog and a ball collide.
* ⚠️ *Note: I do not own the art/UI assets used in this project.*
